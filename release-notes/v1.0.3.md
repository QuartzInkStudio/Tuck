# Tuck 1.0.3

Latest public release for macOS 14+.

## Highlights

- Fixed first-run onboarding so Accessibility and Screen Recording prompts are requested only from the matching user action.
- Prevented macOS permission dialogs from being hidden behind the onboarding window.
- Improved Shelf Mode so Screen Recording is requested when opening Shelf, not during startup.
- Added a blue frosted-glass onboarding background.
- Improved Screen Recording registration so Tuck appears in System Settings.
- Improved menu bar icon scanning and fallback icons.
