# Tuck

**Tuck은 가벼운 macOS 메뉴 막대 관리 앱입니다.** 한 번의 클릭으로 메뉴 막대 아이콘을 숨기고, 노치가 있는 MacBook에서는 Shelf Mode로 숨긴 아이콘을 깔끔하게 보여줍니다.

[다운로드](https://usetuck.com/Tuck-1.0.3.dmg) · [릴리스 노트](https://usetuck.com/releases.html) · [웹사이트](https://usetuck.com/)

> 이 저장소는 Tuck의 공개 다운로드, 릴리스 노트, 제품 정보를 위한 공간입니다. 상용 앱 소스 코드는 공개하지 않습니다.

## 기능

- **Push Mode**: Tuck 왼쪽의 메뉴 막대 아이콘을 한 번에 숨깁니다.
- **Shelf Mode**: 숨긴 아이콘을 메뉴 막대 아래 패널에 표시합니다.
- **Auto-hide / Hover reveal**: 필요할 때만 아이콘을 표시합니다.
- **Per-icon rules**: 아이콘별로 항상 표시, 항상 숨김, 토글을 설정합니다.
- **Shortcut**: `Command + Shift + B`로 어디서든 전환합니다.
- **Zero telemetry**: 분석, 추적, 백그라운드 데이터 수집이 없습니다.

- **피드백을 한곳에 모을 수 있습니다**: 버그, 기능 요청, 호환성 보고, 릴리스 질문이 흩어지지 않습니다.

- **여러 언어를 환영합니다**: 한국어로 작성해도 됩니다. 짧은 영어 요약이 있으면 도움이 되지만 필수는 아닙니다.
