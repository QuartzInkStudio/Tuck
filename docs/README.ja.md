# Tuck

**Tuck は軽量な macOS メニューバー管理アプリです。** ワンクリックでアイコンの clutter を隠し、ノッチ付き MacBook では Shelf Mode で隠したアイコンを見やすく表示します。

> このリポジトリは Tuck の公開ダウンロード、リリースノート、製品情報のためのものです。商用アプリのソースコードは公開していません。

## 主な機能

- **Push Mode**：Tuck の左側にあるメニューバーアイコンをワンクリックで隠します。
- **Shelf Mode**：隠したアイコンをメニューバー下のパネルに表示します。
- **Auto-hide / Hover reveal**：必要な時だけアイコンを表示できます。
- **Per-icon rules**：個別に常時表示、常時非表示、切り替えを設定できます。
- **Shortcut**：`Command + Shift + B` でどこからでも切り替え。
- **No telemetry**：解析、追跡、バックグラウンドデータ収集はありません。
