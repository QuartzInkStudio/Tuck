# Tuck

**Tuck 是一款轻量的 macOS 菜单栏管理工具。** 一键隐藏拥挤的菜单栏图标，Notch MacBook 可使用 Shelf 模式，并且默认不做任何遥测追踪。

[下载 Tuck](https://usetuck.com/Tuck-1.0.8.dmg) · [更新日志](https://usetuck.com/releases.html) · [官网](https://usetuck.com/)

> 这个仓库用于公开发布 Tuck 的下载、更新日志和产品介绍。商业版 App 源码不在此仓库公开。

## 功能

- **Push Mode**：一键隐藏 Tuck 左侧的菜单栏图标。
- **Shelf Mode**：在刘海屏 MacBook 上，把隐藏图标显示在菜单栏下方的面板中。
- **自动隐藏和悬停显示**：平时保持清爽，需要时快速唤出。
- **单图标规则**：可设置始终显示、始终隐藏或跟随切换。
- **全局快捷键**：使用 `Command + Shift + B` 随时切换。
- **零遥测**：不做 analytics、不追踪、不收集后台数据。


## Issue 和反馈

欢迎在这个仓库提交 issue：<https://github.com/QuartzInkStudio/Tuck/issues>

这样做有几个好处：

- **反馈集中**：Bug、功能建议、兼容性问题、版本讨论都集中在一个公开位置，不会散落在邮件里。
- **进度透明**：大家可以看到某个问题是否已确认、正在处理、已修复或已发布。
- **信息更完整**：截图、macOS 版本、Tuck 版本、复现步骤都能保留在同一个线程里。
- **更容易排序优先级**：其他用户可以通过评论和 reaction 表达同样需求，帮助我们判断哪些问题最重要。
- **支持多语言**：可以直接用中文提交；如果方便，附上一句英文摘要会更好，但不是必须。
