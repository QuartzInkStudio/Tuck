# Tuck

**Tuck est un gestionnaire léger de barre des menus pour macOS.** Masquez les icônes en un clic, utilisez Shelf Mode sur les MacBook avec encoche, et gardez une expérience privée par défaut.

[Télécharger](https://usetuck.com/Tuck-1.0.3.dmg) · [Notes de version](https://usetuck.com/releases.html) · [Site web](https://usetuck.com/)

> Ce dépôt contient les téléchargements publics, les notes de version et les informations produit. Le code source commercial de l’app n’est pas publié ici.

## Fonctionnalités

- **Push Mode** : masque les icônes à gauche de Tuck en un clic.
- **Shelf Mode** : affiche les icônes masquées dans un panneau sous la barre des menus.
- **Masquage automatique et survol** : affichez les icônes uniquement quand vous en avez besoin.
- **Règles par icône** : toujours afficher, toujours masquer ou basculer chaque icône.
- **Raccourci global** : `Command + Shift + B`.
- **Zéro télémétrie** : pas d’analytics, pas de suivi, pas de collecte en arrière-plan.
